5. Дан код:
class A {
    public function foo() {
        static $x = 0;
        echo ++$x;
    }
}
$a1 = new A(); //Создаёт объект.
$a2 = new A(); //Создаёт объект.
$a1->foo(); //Вызывает функцию и выводит 1.
$a2->foo(); //Вызывает функцию и выводит 2.
$a1->foo(); //Вызывает функцию и выводит 3.
$a2->foo(); //Вызывает функцию и выводит 4.
Что он выведет на каждом шаге? Почему?
Потому что $х static иначе всё выводы бы заново делали $х 0 и выводился бы 1 всегда.
